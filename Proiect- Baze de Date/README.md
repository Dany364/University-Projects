# 🏨 Platforma pentru Rezervari Hoteluri si Restaurante

![Schema](Schema.png)

---

# 📖 Descriere

Acest proiect a fost realizat in cadrul disciplinei **Baze de Date** din cadrul Facultatii de Electronica, Telecomunicatii si Tehnologia Informatiei.

Proiectul prezinta proiectarea unei baze de date pentru o platforma software destinata gestionarii rezervarilor la hoteluri si restaurante. Scopul aplicatiei este de a oferi utilizatorilor posibilitatea de a realiza, dintr-o singura platforma, rezervari pentru cazare si, optional, pentru restaurantele partenere.

Pe langa definirea structurii bazei de date, proiectul include modelarea entitatilor si relatiilor dintre acestea, definirea cheilor primare si externe, optimizarea prin indecsi si realizarea dictionarului de date.

---

# 🎯 Obiective

- Proiectarea unei baze de date relationale.
- Eliminarea relatiilor de tip Many-to-Many prin introducerea entitatii **Perioade**.
- Gestionarea rezervarilor pentru hoteluri si restaurante.
- Organizarea datelor pe locatii geografice.
- Stocarea istoricului rezervarilor si recenziilor.
- Optimizarea interogarilor prin utilizarea cheilor si a indecsilor.

---

# 🗄️ Entitati principale

Baza de date contine urmatoarele entitati:

- Clienti
- Hoteluri
- Camere
- Restaurante
- Mese
- RezervariHotel
- RezervariRestaurant
- Perioade
- Recenzii
- Locatii

Aceste entitati sunt conectate prin relatii one-to-many si many-to-one, folosind chei primare si chei externe pentru mentinerea integritatii datelor.

---

# ⚙️ Functionalitati

- Gestionarea utilizatorilor.
- Rezervari hoteliere.
- Rezervari la restaurante.
- Gestionarea camerelor si meselor disponibile.
- Gestionarea locatiilor.
- Publicarea recenziilor si acordarea unui rating.
- Evidenta perioadelor de rezervare.
- Evitarea suprapunerii rezervarilor.
- Pastrarea istoricului rezervarilor pentru analiza si raportare.

---

# 🧩 Elemente de proiectare

In cadrul proiectului au fost realizate:

- analiza cerintelor;
- identificarea entitatilor si atributelor;
- stabilirea relatiilor dintre tabele;
- diagrama Entitate-Relatie (ER);
- dictionarul de date;
- definirea cheilor primare si externe;
- propunerea indecsilor pentru optimizarea bazei de date.
